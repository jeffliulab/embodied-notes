    • Subscriber: subscribe topic
    • Broker: дёӯиҪ¬(еҝ…йңҖ)

    3. QoS (Quality of Service)

    Level еҗ«д№ү з”ЁдҫӢ

    1 At least once (ACK) йҮҚиҰҒж•°жҚ®(е…Ғи®ёйҮҚеӨҚ)

    QoS и¶Ҡй«ҳ,delay + overhead и¶ҠеӨ§гҖӮ

    7. Python вҖ” paho-mqtt

    import paho.mqtt.client as mqtt

def on_connect(client, userdata, flags, rc):
    print("Connected:", rc)
    client.subscribe("home/+/temperature", qos=1)

def on_message(client, userdata, msg):
    print(f"{msg.topic}: {msg.payload.decode()}")

client = mqtt.Client()
client.on_connect = on_connect
client.on_message = on_message
client.username_pw_set("user", "pass")
client.will_set("status/device01", "offline", qos=1, retain=True)
client.connect("broker.hivemq.com", 1883, keepalive=60)
client.loop_forever()
