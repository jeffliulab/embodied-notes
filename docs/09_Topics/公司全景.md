# 具身智能公司全景

## 概述

具身智能（Embodied Intelligence）正处于从实验室走向产业化的关键阶段。全球范围内，数十家公司正在开发人形机器人、四足机器人、灵巧操作系统和基础模型平台。本文梳理全球及中国主要公司的技术路线、融资状态和产品进展。

### 重点公司详解

#### Tesla Optimus

Tesla的人形机器人项目于2021年AI Day公布，2022年展示原型，2024-2025年在Fremont和德州工厂进行内部测试部署。核心优势：

- **数据飞轮**：FSD积累的视觉理解能力可迁移到机器人
- **垂直整合**：自研芯片（Dojo/HW5）、电机、电池、推理硬件
- **规模化制造**：汽车工厂的大规模制造经验
- **成本目标**：Elon Musk声称最终售价可低于$20,000

#### Figure

Figure是近年增长最快的人形机器人公司：

- **Figure 01**：首款原型，展示了基本的行走和抓取能力
- **Figure 02**：第二代产品，集成OpenAI语言模型实现语音交互
- **技术路线**：端到端VLA模型 + 全身运动控制
- **商业路线**：与BMW合作在Spartanburg工厂进行汽车装配试点

#### Physical Intelligence

2024年成立即获得巨额融资，创始团队来自Google DeepMind和UC Berkeley：

- **pi0**：首个通用机器人基础模型，基于VLA架构
- **pi0.5**：改进版，支持更多任务和机器人平台
- **技术理念**：用互联网规模数据预训练 + 机器人数据微调
- **核心团队**：Karol Hausman、Sergey Levine等知名研究者

### 中国公司特点分析

#### 第一梯队：已有量产产品

- **宇树科技**：中国四足机器人领导者，Go2售价仅$1,600起，H1/G1人形机器人具备高动态运动能力
- **优必选**：首家人形机器人上市公司，Walker系列已在多个场景试点
- **傅利叶智能**：康复机器人起家，GR系列人形机器人强调力控和安全交互

#### 第二梯队：快速崛起的新势力

- **智元机器人**：2023年成立，创始人来自华为，融资速度和产品迭代极快
- **银河通用**：专注灵巧操作，通用抓取技术路线
- **星动纪元**：清华大学背景，高动态全身运动控制

#### 垂直赛道领先者

- **云深处科技**：四足机器人在电力巡检、应急救援等场景已商业化
- **思灵机器人**：力控协作臂技术来自DLR，在精密装配领域有优势

## 竞争格局分析

### 技术路线对比

| 维度 | 美国公司 | 中国公司 | 欧洲公司 |
|------|----------|----------|----------|
| **AI能力** | 领先（大模型生态） | 快速追赶 | 学术积累深厚 |
| **硬件成本** | 较高 | 显著优势 | 中等 |
| **制造能力** | Tesla领先 | 供应链完整 | 精密制造 |
| **商业落地** | 工厂试点为主 | 多场景探索 | 工业应用 |
| **融资环境** | 大额融资活跃 | 政策+资本双驱动 | 相对保守 |

### 发展趋势

1. **大模型赋能**：几乎所有公司都在集成VLA等大模型技术
2. **成本战争**：中国公司在硬件成本上有显著优势，推动全球价格下降
3. **场景聚焦**：从通用走向特定场景（工厂、物流、巡检）的聚焦
4. **生态竞争**：NVIDIA（Isaac/GR00T）、Google（RT-X）等平台生态之争
5. **政策驱动**：中国将人形机器人列为战略新兴产业，多地出台支持政策

### 投资热度

2023-2025年，具身智能领域投资呈现爆发式增长：

- **全球**：Figure、Physical Intelligence、1X等公司融资总额超过$20亿
- **中国**：智元、银河通用、星动纪元等新公司成立即获大额融资
- **战略投资**：OpenAI投资1X和Figure，Microsoft投资Sanctuary，Hyundai收购Boston Dynamics
