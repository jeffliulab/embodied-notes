# 具身智能关键会议与期刊

## 概述

具身智能是一个高度交叉的领域，横跨机器人学、计算机视觉、机器学习和人工智能。本文整理该领域最重要的学术会议和期刊，包括投稿建议、接收率趋势、近年最佳论文方向，帮助研究者选择合适的发表渠道。

---

## 1. 核心会议

### 1.1 机器人学习专属会议

#### CoRL（Conference on Robot Learning）

| 属性 | 信息 |
|------|------|
| 创办年份 | 2017 |
| 频率 | 每年一次 |
| 接收率 | ~30%（2023: 31%，2024: 28%） |
| 审稿方式 | 双盲同行评审 |
| 核心主题 | 机器人学习、模仿学习、强化学习与机器人 |

CoRL 是具身智能领域最聚焦的顶级会议，论文质量极高。近年来已成为发布 VLA 模型、扩散策略等前沿工作的首选会议。

**近年最佳论文趋势**：

- 2022: Diffusion Policy（扩散模型用于机器人策略）
- 2023: 跨具身迁移学习
- 2024: 基础模型驱动的机器人操作

#### RSS（Robotics: Science and Systems）

| 属性 | 信息 |
|------|------|
| 创办年份 | 2005 |
| 频率 | 每年一次 |
| 接收率 | ~25-30% |
| 审稿方式 | 双盲，多轮审稿 |
| 核心主题 | 机器人学全领域，偏重理论和算法 |

RSS 规模较小但论文质量公认顶级，审稿流程严谨。偏好有扎实理论支撑的工作。

### 1.2 大型机器人会议

#### ICRA（IEEE International Conference on Robotics and Automation）

| 属性 | 信息 |
|------|------|
| 创办年份 | 1984 |
| 频率 | 每年一次 |
| 接收率 | ~40-45% |
| 投稿量 | 3000-4000 篇/年 |
| 核心主题 | 机器人学与自动化全领域 |

ICRA 是全球最大的机器人学会议，覆盖面最广。适合发表系统级工作、工程创新和应用研究。

#### IROS（IEEE/RSJ International Conference on Intelligent Robots and Systems）

| 属性 | 信息 |
|------|------|
| 创办年份 | 1988 |
| 频率 | 每年一次 |
| 接收率 | ~40-47% |
| 投稿量 | 2500-3500 篇/年 |
| 核心主题 | 智能机器人系统 |

IROS 与 ICRA 规模相当，更偏重智能系统和人机交互方向。

#### Humanoids（IEEE-RAS International Conference on Humanoid Robots）

| 属性 | 信息 |
|------|------|
| 频率 | 每年一次 |
| 规模 | 中小型 |
| 核心主题 | 人形机器人运动控制、仿人感知与交互 |

随着人形机器人热潮，该会议关注度大幅提升。

### 1.3 AI/ML 顶会中的机器人方向

#### NeurIPS / ICML / ICLR

这三大 ML 顶会近年来接受了越来越多的具身智能论文：

| 会议 | 接收率 | 机器人相关 Workshop |
|------|--------|-------------------|
| NeurIPS | ~25% | Robot Learning Workshop, Foundation Models for Robotics |
| ICML | ~25% | Aligning Robot and Human Representations |
| ICLR | ~30% | Generalization in Planning |

**趋势**：具身智能的"学习"部分（VLA、扩散策略、世界模型）越来越多地发表在 ML 顶会，而"机器人系统"部分仍以 ICRA/IROS/CoRL 为主。

#### CVPR / ECCV / ICCV

计算机视觉顶会中具身相关方向：

- 3D 场景理解与重建
- 具身导航（Embodied Navigation）
- 视觉-语言 Grounding
- 仿真环境基准测试（Habitat、AI2-THOR）

## 2. 核心期刊

### 2.1 IEEE 期刊

#### RA-L（IEEE Robotics and Automation Letters）

| 属性 | 信息 |
|------|------|
| 影响因子 | ~5.2 |
| 审稿周期 | 3-6 个月 |
| 特点 | 可选择在 ICRA/IROS 做口头报告 |

RA-L 是目前机器人领域发表效率最高的期刊，与 ICRA/IROS 联合审稿机制使其非常受欢迎。

#### T-RO（IEEE Transactions on Robotics）

| 属性 | 信息 |
|------|------|
| 影响因子 | ~7.8 |
| 审稿周期 | 6-12 个月 |
| 特点 | 机器人领域最顶级期刊之一 |

T-RO 要求论文有完整的理论分析和充分的实验验证，适合发表成熟的系统性工作。

### 2.2 综合性顶级期刊

#### IJRR（International Journal of Robotics Research）

| 属性 | 信息 |
|------|------|
| 影响因子 | ~7.5 |
| 创办年份 | 1982 |
| 特点 | 机器人学历史最悠久的顶刊 |

IJRR 偏好深入、完整的研究贡献，审稿严格但公正。

#### Science Robotics

| 属性 | 信息 |
|------|------|
| 影响因子 | ~25 |
| 创办年份 | 2016 |
| 特点 | Science 子刊，影响力极高 |

Science Robotics 关注突破性进展和具有广泛影响的机器人研究，对于提升具身智能领域的社会影响力至关重要。

## 4. 最佳论文趋势分析（2022-2025）

### 4.1 关键趋势

**2022: 大规模机器人学习元年**

- RT-1 展示了大规模真实数据训练的价值
- Diffusion Policy 引入生成模型范式
- SayCan 将 LLM 引入机器人任务规划

**2023: VLA 模型涌现**

- RT-2 证明了 VLM $\rightarrow$ VLA 迁移的可行性
- 多个仿真基准平台发布
- 灵巧操作研究复兴

**2024: 基础模型与开放生态**

- Open X-Embodiment 推动跨具身数据共享
- pi0 等通用策略模型出现
- 人形机器人大量产业投资

**2025: 世界模型与具身推理**

- 视频生成模型用于机器人规划
- 具身推理和长程规划成为焦点
- Sim-to-Real 差距进一步缩小

### 4.2 热门关键词

近三年论文关键词频率变化：

| 关键词 | 2022 | 2023 | 2024 |
|--------|------|------|------|
| Foundation Model | ++ | +++ | ++++ |
| Diffusion Policy | + | +++ | ++++ |
| VLA | - | ++ | ++++ |
| World Model | + | ++ | +++ |
| Sim-to-Real | +++ | +++ | +++ |
| Dexterous Manipulation | ++ | +++ | ++++ |
| Humanoid | + | ++ | ++++ |

## 5. 重要 Workshop 和竞赛

### 5.1 Workshop

- **NeurIPS Robot Learning Workshop**：每年最前沿的方向讨论
- **ICRA Workshops**：涵盖具身 AI 各子方向
- **CoRL Workshop on Pre-training for Robotics**：基础模型在机器人中的应用

### 5.2 竞赛/Benchmark

- **SAPIEN ManiSkill**：操作技能评估
- **Habitat Challenge**：具身导航
- **RoboCup**：多机器人协作
- **DARPA Challenges**：极端环境机器人
