# 机器人工程

机器人工程（Robot Engineering）聚焦机器人各子系统的**工程实践**——从计算机体系结构到传感器、执行器、通信、电源、机械结构的完整技术栈。本板块以硬件工程师/Maker视角，深入每个子系统的工作原理、接口协议、代表性产品拆解和实际编程示例。

以人形机器人、四足机器人、扫地机器人、机械臂等主流形态为参照，确保覆盖所有关键工程子系统。

**本板块内容：**

- **[计算机工程](01_Computer_Engineering/index.md)** — CPU/GPU架构、内存、总线、嵌入式系统、操作系统
- **[相机与视觉](02_Cameras/index.md)** — RGB/深度/双目/事件相机、标定
- **[激光雷达](03_LiDAR/index.md)** — 2D/3D/固态LiDAR、点云处理
- **[惯性导航与定位](04_IMU_Navigation/index.md)** — IMU、VIO、GPS/RTK、近距离传感器
- **[电机与执行器](05_Motors_Actuators/index.md)** — DC/BLDC/舵机、减速器、末端执行器
- **[通信与总线](06_Communication/index.md)** — UART/I2C/SPI/CAN/EtherCAT/WiFi
- **[电源管理](07_Power/index.md)** — 电池、BMS、DC-DC、功耗预算
- **[机械结构](08_Mechanical/index.md)** — 底盘、机械臂、散热防护、3D打印、CAD
- **[显示与人机交互](09_Displays_HMI/index.md)** — OLED/LCD、Web界面、遥控手柄
- **[力觉与触觉传感](10_Force_Tactile/index.md)** — 六维力矩、触觉皮肤、足端传感、力控
- **[系统集成](11_Integration/index.md)** — PCB设计、ROS2集成、调试、机器人拆解案例
